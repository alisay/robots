## To install: 

```bash
npm install
```

## To use: 

run the program using the command 

```bash
npm start 
```

The program reads in commands from the input.txt file in the following form:

    PLACE X,Y,F
    MOVE
    LEFT
    RIGHT
    REPORT 

## To test

install jest from command line with:

```bash
npm install jest
```
then run the tests using:

```bash
npm test
```
