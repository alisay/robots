## To install: 

```bash
npm install
```

## To use: 

run the program using the command 

```bash
start ./input.txt
```

where the input file contains commands of the following form:

    PLACE X,Y,F
    MOVE
    LEFT
    RIGHT
    REPORT 

## To test

install jest from command line with:

```bash
npm install jest --global
```
then run the tests using:

```bash
npm run test
```
